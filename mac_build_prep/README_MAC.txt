# YouTube Downloader macOS 懶人打包指南

您只需做三個動作，就能在 Mac 上獲得此應用程式：

1. **複製**：
   將 `mac_build_prep` 整個資料夾複製到您的 Mac 電腦上。

2. **執行**：
   在 Mac 上進入該資料夾，找到 `build.command` 檔案。
   - **點擊兩下** `build.command` 執行它。
   - (如果因為權限問題無法直接執行，請在終端機輸入 `chmod +x build.command` 然後再執行)

3. **完成**：
   腳本跑完後，會自動彈出一個視窗，裡面的 `YouTubeDownloader` 就是您的應用程式了！

---
注意：
- 執行過程中需要網路連線下載必要元件。
- 第一次開啟應用程式時，如果 Mac 提示「無法打開未識別開發者的程式」，請到「系統設定」>「隱私權與安全性」中點擊「強制打開」。
